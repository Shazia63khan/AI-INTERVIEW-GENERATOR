# AI Interview Question Generator 🤖

A full-stack web app that generates AI-based interview questions using FastAPI, React (Vite), and Groq LLM API.

## Tech Stack
- React + Vite
- FastAPI
- SQLite
- Groq API

## Features
- AI-generated interview questions
- Topic-based generation
- Save & view history

## Run Locally

Backend:
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

Frontend:
cd frontend
npm install
npm run dev
